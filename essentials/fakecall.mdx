Esse bloco documenta como chamar a atenção do seu cliente através de uma ligação simulada.

Para realizar uma ligação simulada, utilize a função **ApiEuAtendo1.FazerLigacao**. A função requer o número de telefone do contato e a duração da chamada em segundos, retornando uma string que contém o ID da ligação criada.

## Passo a Passo para Fazer a Ligação Simulada

1. Insira o **Número de Telefone** do contato para o qual deseja realizar a ligação.
2. Insira a **Duração** da ligação em segundos.
3. Chame a função **FazerLigacao** passando o número e a duração.
4. A função retornará um ID único da ligação, que pode ser usado para referência futura.

### Exemplo de Código

```delphi
// Chamar a função para realizar a ligação simulada
procedure TForm9.RealizarLigacaoSimulada;
var
  IDLigacao: string;
begin
  // Chamar a função passando o número de telefone e a duração da chamada
  IDLigacao := ApiEuAtendo1.FazerLigacao(edtNumeroTelefone.Text, StrToInt(edtDuracao.Text));
  
  // Exibir o ID retornado
  ShowMessage('ID da ligação simulada: ' + IDLigacao);
end;
```

## Parâmetros da Função

- `edtNumeroTelefone.Text`: Número de telefone do contato que receberá a ligação.
- `edtDuracao.Text`: Duração da ligação simulada em segundos.

## Resultado Esperado
A função retornará uma string contendo o ID da ligação criada. Esse ID pode ser utilizado para rastreamento ou referência.

## Tratamento de Erros
Caso o número de telefone ou a duração sejam inválidos, a função pode retornar uma mensagem de erro. Verifique se os parâmetros foram inseridos corretamente antes de chamar a função.
