    ### Funcionalidades Disponíveis

    - **Obter Dados do Contato**
    - **Obter Foto de Perfil do Contato**
    - **Verificar se o Número é WhatsApp Válido**

    ## Obter Dados do Contato

    Para obter informações detalhadas de um contato, como nome e status, utilize a função **ApiEuAtendo1.ObterDadosDoContato**, passando o número do contato como parâmetro.

    ### Exemplo de Código

    ```delphi
    // Chamar a função para obter dados de um contato
    ApiEuAtendo1.ObterDadosDoContato(edtNumeroContato.Text);

    // Evento disparado ao obter os dados do contato
    procedure TForm9.ApiEuAtendo1ObterDadosDoContato(Sender: TObject;
      const DadosContato: TContactInfo);
    begin
      memo1.Lines.Add('Nome: ' + DadosContato.ContactName);
      memo1.Lines.Add('Status: ' + DadosContato.Status);
      memo1.Lines.Add('Número: ' + DadosContato.PhoneNumber);
      memo1.Lines.Add('--------------------------------------');
    end;
    ```

    ## Obter Foto de Perfil do Contato

    Para obter a foto de perfil de um contato, utilize a função **ApiEuAtendo1.ObterFotoDePerfil** passando o número do contato. Isso retornará a URL da foto de perfil do contato no WhatsApp.

    ### Exemplo de Código

    ```delphi
    // Chamar a função para obter a foto de perfil de um contato
    ApiEuAtendo1.ObterFotoDePerfil(edtNumeroContato.Text);

    // Evento disparado ao obter a foto de perfil do contato
    procedure TForm9.ApiEuAtendo1ObterFotoDePerfil(Sender: TObject;
      const FotoPerfilURL: string);
    begin
      memo1.Lines.Add('URL da Foto de Perfil: ' + FotoPerfilURL);
      // Exemplo: exibir a foto de perfil em um componente de imagem
      Image1.Picture.LoadFromFile(FotoPerfilURL);
    end;
    ```

    ## Verificar se o Número é WhatsApp Válido

    Para verificar se um número é válido no WhatsApp, utilize a função **ApiEuAtendo1.VerificarNumeroWhatsApp** passando o número do contato como parâmetro. A função retorna uma resposta indicando se o número é válido ou não.

    ### Exemplo de Código

    ```delphi
    // Chamar a função para verificar se o número é um WhatsApp válido
    procedure TForm9.VerificarNumeroWhatsApp;
    var
      IsWhatsAppValido: Boolean;
      ErrorMsg: string;
    begin
      IsWhatsAppValido := ApiEuAtendo1.VerificarNumeroWhatsApp(edtNumeroContato.Text, ErrorMsg);
      
      if IsWhatsAppValido then
        ShowMessage('O número é válido no WhatsApp.')
      else
        ShowMessage('Número inválido: ' + ErrorMsg);
    end;
    ```

    ## Tratamento de Erros
    - Verifique se o número de telefone foi inserido corretamente antes de chamar qualquer uma das funções.
    - Em caso de erro, as funções retornarão uma mensagem de erro específica através da variável `ErrorMsg`, para facilitar o diagnóstico do problema.
