## Instalação do Componente euAtendo

Para instalar o componente *euAtendo*, siga os passos abaixo:

### 1. Baixar e copiar os arquivos

1. Baixe os arquivos do repositório oficial.
2. Copie os arquivos para um diretório de sua preferência no seu sistema.

### 2. Instalar o pacote no Delphi

1. Abra o **Delphi**.
2. Navegue até a pasta onde os arquivos foram salvos:
   - Acesse as pastas `build` e depois `install`.
3. Prossiga para instalar o pacote clicando no arquivo correspondente.

### 3. Adicionar o caminho das units ao Delphi

1. No Delphi, vá em **Tools > Options...**.
2. Em **Environment Options > Delphi Options > Library**, adicione o caminho onde os arquivos do *euAtendo* estão localizados.

   > **Dica:** Certifique-se de adicionar o caminho correto da pasta para garantir que todas as units sejam localizadas corretamente durante o desenvolvimento.
