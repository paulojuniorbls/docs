## Pré-requisitos

Antes de começar, certifique-se de que você atenda aos seguintes requisitos:

- Delphi instalado (versão compatível)
- Componente euAtendo baixado (disponível em nossa página oficial)
- Acesso ao seu painel euAtendo para obter as credenciais de API

## Passo 1: Baixando o Componente

Faça o download da versão mais recente do componente *euAtendo* diretamente do nosso repositório ou site oficial. 

**Link para download**: [Baixar Componente](https://seusite.com/download-euatendo)

## Passo 2: Instalando no Delphi

1. Abra seu Delphi e acesse o menu **Component > Install Package**.
2. Clique em **Add...** e navegue até o diretório onde você extraiu o componente.
3. Selecione o arquivo `.dpk` do componente *euAtendo* e clique em **OK**.
4. Confirme a instalação e verifique se o componente foi adicionado à paleta de componentes.

> **Dica:** Se ocorrer algum erro durante a instalação, verifique se todas as dependências estão corretamente configuradas no Delphi.

## Passo 3: Configuração Inicial

Após a instalação, você pode começar a configurar o *euAtendo* em seu projeto Delphi.

1. Arraste o componente *euAtendo* para o seu formulário.
2. Insira suas credenciais da API (chave de API e Token) que podem ser obtidas no painel do *euAtendo*.
3. Configure os parâmetros desejados, como URL de integração, filas e usuários.

```delphi
euAtendo1.APIKey := 'sua-chave-api';
euAtendo1.Token := 'seu-token';
euAtendo1.Fila := 'nome-da-fila';
