Esse bloco documenta como validar se um número possui um WhatsApp ativo utilizando o método **ApiEuAtendo1.ExistWhats**. Ele explica como passar os parâmetros corretos e exibe um exemplo de código Delphi para realizar a validação e exibir o resultado.

Para verificar se um número possui um WhatsApp válido, utilize a função **ApiEuAtendo1.ExistWhats**. A função requer o número do telefone e retornará um valor booleano indicando se o número está registrado no WhatsApp, além de uma mensagem de erro, se houver.

## Passo a Passo para Validar o Número

1. Insira o **Número de Telefone** no campo correspondente.
2. Chame a função **ExistWhats** passando o número.
3. A função retornará `True` se o número estiver registrado no WhatsApp e `False` caso contrário.
4. Em caso de erro, a função também preencherá a variável com uma mensagem indicando o motivo do erro.

### Exemplo de Código

```delphi
// Chamar a função para validar se o número possui WhatsApp
procedure TForm9.ValidarNumeroWhatsApp;
var
  Numero, ErroMsg: string;
  Existe: Boolean;
begin
  Numero := edtNumeroContato.Text; // Assume que edtNumero é um TEdit onde o número é inserido
  try
    Existe := ApiEuAtendo1.ExistWhats(Numero, ErroMsg);
    if Existe then
      memo1.Lines.add('O número ' + Numero + ' possui WhatsApp.')
    else if ErroMsg <> '' then
      memo1.Lines.add('Erro ao verificar o número: ' + ErroMsg)
    else
      memo1.Lines.add('O número ' + Numero + ' não possui WhatsApp.');
  except
    on E: Exception do
    begin
      memo1.Lines.add('Erro ao verificar o número: ' + E.Message);
    end;
  end;
end;
```

## Parâmetros da Função

- `edtNumeroContato.Text`: Número de telefone que você deseja validar.

## Resultado Esperado
A função retornará um valor booleano indicando se o número está registrado no WhatsApp:
- **True**: O número possui WhatsApp.
- **False**: O número não possui WhatsApp.

Além disso, uma mensagem de erro pode ser retornada caso haja algum problema durante a verificação.

## Tratamento de Erros
Caso o número de telefone não seja válido ou ocorra algum problema durante a verificação, a função pode retornar uma mensagem de erro. Verifique se o número foi inserido corretamente antes de chamar a função.
